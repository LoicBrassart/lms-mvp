Lors de la conception d'une interface, certains éléments de style vont être récurrents. Les polices de la charte graphique vont par exemple être appliqués sur l'ensemble de vos designs. Les styles vont vous permettre de conserver une cohérence sur l'ensemble de l'aspect des éléments.

## Qu'est-ce que les styles ?

Les styles permettent de définir certaines propriétés que l'on peut ensuite réutiliser et appliquer à plusieurs éléments. Plutôt que de paramétrer de nouveau chaque élément ajouté à vos designs, les styles nous permettent de leur appliquer rapidement ces paramètres, définis en une fois.

Il est possible de créer des styles pour les couleurs, le texte, les effets et les grilles de mise en page. Le style peut être créé pour les propriétés suivantes :

- **Couleur :** remplissage, contour, couleur de fond
- **Texte :** famille de police, taille, interligne, crénage
- **Effets :** flou de calque, flou de fond, ombre portée, ombre interne
- **Grilles :** colonnes, rangées, grille

## Créer et appliquer des styles

Comme vu précédemment, il est possible d'appliquer une variété de styles aux éléments de notre design. La démarche à suivre est similaire pour l'ensemble des types de style :

- Sélectionnez l'objet (ou la frame dans le cas d'une grille) pour lequel vous souhaitez créer un style en lui appliquant les attributs souhaités
- Cliquez sur l'icône "Style" (4 ronds qui forment un carré) situé sur la panneau propriétés à droite. Cliquez ensuite sur l'icône "Create style" (+). Vous devez vous rendre dans la section appropriée. Pour un style de couleur par exemple, nous irions créer notre style dans la section "Fill", "Stroke" ou "Background Color".
- Nommez votre style puis cliquez sur le bouton "Create style"

Nous appliquons certains attributs à un calque de texte (police de caractères, graisse, taille) pour que celui-ci ait l'aspect souhaité. Nous créons un style d'après celui-ci en allant sur la section « Text » du panneau de droite, puis en cliquant sur l'icône « Style » et enfin sur l'icône « Create style » présente sur la fenêtre des styles de texte (« Text styles »).

Lorsque votre style est créé, vous pouvez l'appliquer à d'autres éléments. Pour ce faire, sélectionnez l'élément auquel vous souhaitez appliquer le style et cliquez sur l'icône "style" de la section voulue. Pour appliquer un style de couleur à une forme par exemple, il suffira de se rendre dans la section "Color" du panneau de droite, de cliquer sur l'icône "Style" puis de choisir le style souhaité dans ceux existants.

## Editer des styles

Vous pouvez éditer les styles créés via le panneau de droite, en passant votre souris sur le style à modifier et en cliquant sur l'icône « Edit style ».

L'exemple ci-dessus reprend le style de texte créé précédemment. Nous pourrions modifier les propriétés appliquées par le biais du panneau Edit style.

Il est également possible de détacher un élément d'un style qui lui était appliqué. Pour ce faire, sélectionnez l'élément que vous souhaitez libérer du style, puis cliquer sur l'icône « Detach style » situé à droite du style appliqué. Cette icône apparaît au survol de la souris sur le style.

## Styles locaux et librairies partagées

### Les styles locaux

Les styles locaux sont les styles présents dans votre fichier. Ces styles sont accessibles via le panneau de droite quand aucun élément n'est sélectionné, ou à partir de l'icône style présente dans les sections.

### Partager et réutiliser des styles dans d'autres projets

Les styles que vous avez créés peuvent être utilisés au sein d'autres projets et partagés avec vos collaborateurs au sein d'une librairie.

Pour ce faire, rendez-vous dans le panneau "Asset" situé sur le panneau gauche et cliquez sur l'icône librairie (livre ouvert). Cliquez ensuite sur le bouton "Publish" de la section "Current file". Vous pourrez alors entrer une description sur les changements effectués si besoin, ainsi que désélectionner certains styles que vous ne souhaiteriez pas voir publiés. Si vous n'avez pas besoin d'effectuer ces actions, ou les avez déjà réalisées, vous pouvez cliquer sur "Publish styles".

## ☝️ Résumé

- Les styles permettent d'**homogénéiser le design** au sein des différents écrans d'un projet et sont un **gain de temps** pour les designers dans leur processus de conception.
- **Les styles peuvent être partagés** dans différents projets d'équipe et accessible aux collaborateurs de ces projets.

## 💪 Challenge

Web designer dans une agence de design, vous épaulez un collègue sur la conception d'un site e-commerce. Celui-ci vous demande de préparer le terrain en créant des styles, afin d'organiser au mieux le fichier et d'appliquer correctement la charte graphique prédéfinie.
