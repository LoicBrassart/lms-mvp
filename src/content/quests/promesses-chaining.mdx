## Contexte

Tu disposes d'une fonction `fetchUser(id)` et d'une fonction `fetchOrders(userId)`, toutes
deux basées sur des callbacks. Réécris le code suivant en utilisant des Promises
chaînées.

## Code de départ

```js
fetchUser(1, (err, user) => {
  if (err) return console.error(err);
  fetchOrders(user.id, (err, orders) => {
    if (err) return console.error(err);
    console.log(orders);
  });
});

Objectif

Réécrire ce code sans callbacks imbriqués, en utilisant .then() et .catch().

```
